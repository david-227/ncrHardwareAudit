#NCRHWAUDIT

##Use
NCRHWAUDIT is a series of interconnected PS Scripts and Batch Files that will pull valuable information regarding the NCR ATM/ITM Hardware attached to a specific machine.

###Current Functions
Currently there are scripts in place that can individually pull the following:
- Hardware ID
- EPP Informatiom
- Depositor information
- ATM/ITM Model
- ATM PC Core Type
- Terminal ID

###Future Features
Currently there are several features roadmapped for the near future:
- SPS Antiskimming Informatiom
- Card Reader information
- Touch Screen information
- ATM Patches / Windows Patches

These scripts have been researched and created by David Thomas on behalf of QSI Banking.
